Documentation Guide

This template lets you build documentation pages using two JavaScript objects: SETTINGS and CONTENT.

Features
  • Background customization
  • Optional disclaimer
  • Headers, text, images, embeds, slideshows

Quick Start

  1. Open the HTML file.
  2. Edit SETTINGS.
  3. Add objects to CONTENT.
  4. Save and refresh.

SETTINGS

const SETTINGS={
 background:"#ffffff",
 disclaimer:{
  enabled:true,
  title:"Disclaimer",
  description:"Message",
  titleColor:"#ff4444",
  buttonColor:"#4CAF50",
  buttonText:"OK"
 }
};

background: CSS background value.

disclaimer.enabled: Shows or hides the popup.

CONTENT

CONTENT is an array. Each object represents one section.

const CONTENT=[];

Supported Types

Header

{type:"header",text:"Welcome",color:"#06f"}

Text

{type:"text",text:"Paragraph",color:"#333"}

Image

{type:"image",src:"photo.png",caption:"Caption",color:"#666"}

Embed

{type:"embed",html:`<iframe src="https://example.com"></iframe>`}

Slideshow

{type:"slideshow",images:["1.png","2.png"]}

Build Your First Page

const CONTENT=[
{type:"header",text:"My Website",color:"#0066ff"},
{type:"text",text:"Welcome!",color:"#222"},
{type:"image",src:"banner.png",caption:"Banner"},
{type:"slideshow",images:["1.png","2.png","3.png"]}
];

Tips

  • Never forget to add "," after every ]} except the very last one.
  • Embed custom HTML with embed and keep your JS out of the embed but at the very end of </script>.
  • Keep image sizes optimized.
